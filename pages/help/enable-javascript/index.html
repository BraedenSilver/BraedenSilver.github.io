Enable JavaScript & Clear Your Cache

Follow the steps below to turn on JavaScript in your browser, then refresh this page. After you enable JavaScript, clear your cache so the newest version of each page loads.

Google Chrome

  1. Open the Chrome menu and choose Settings.
  2. Select Privacy and security → Site Settings.
  3. Under Content, open JavaScript and switch it to Sites can use JavaScript.
  4. Return to Privacy and security and choose Clear browsing data. Clear cached images and files.
  5. Reload this page.

Mozilla Firefox

  1. Type about:config in the address bar and press Enter.
  2. Accept the warning if prompted.
  3. Search for javascript.enabled and set it to true.
  4. Open the Firefox menu → Settings → Privacy & Security, then scroll to Cookies and Site Data and click Clear Data… to remove cached content.
  5. Reload this page.

Safari (macOS)

  1. Open Safari → Settings.
  2. Go to the Advanced tab and ensure Show features for web developers is enabled if needed.
  3. In the Advanced tab, confirm that Enable JavaScript is checked.
  4. Back in the main menu, choose Safari → Clear History… and clear cached files.
  5. Reload this page.

Need More Help?

If you're using a different browser, search its support center for “enable JavaScript” and “clear cache.” Once you're done, revisit the page you were trying to open.
